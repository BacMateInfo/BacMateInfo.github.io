    Progresii aritmetice

    Definitie: Se numeste progresie aritmetica un sir de numere reale in care fiecare termen se obtine din termenul anterior adunand o constanta numita ratie (r)

    Proprietate: Trei numere a , b, c sunt in progresie aritmetica daca b e medie aritmetica intre a si c adica $$b = {a + c \over 2} $$

    $$a_n=a_1+(n - 1)r$$

    $$S_n = \frac{n(2a_1 + (n - 1)r)}{2}$$ $$S_n = {a_1 + a_2 + a_3 + ... + a_n}$$

    Progresii geometrice

    Definitie: Se numeste progresie geometrica un sir de numere reale in care fiecare termen se obtine din termenul anterior inmultind cu o constanta numita ratie (q).

    Proprietate: Trei numere a , b, c sunt in progresie geometrica cu termeni pozitivi daca b e medie geometrica intre a si c adica \(b = \sqrt{ac}\)

    $$a_n=a_1q^n - 1$$

    $$S_n = \frac{a_1(q^n-1)}{q - 1}$$ $$S_n = a_1 + a_2 + a_3 + ... + a_n$$

    Elemente de combinatorica

    Permutari de n se noteaza \(P_n\)
    \(P_n=n!\) si reprezinta numarul de multimi ordonate ce se pot forma cu n elemente.

    Aranjamente de n luate cate k se noteaza \(A_n^k\)
    \(A_n^k = \cfrac{n!}{(n-k)!}\) reprezinta numarul de submultimi ordonate de cate k elemente ce se pot forma dintr-o multime de n elemente.

    Combinari de n luate cate k se noteaza \(C_n^k\)
    \(C_n^k = \cfrac{n!}{k!(n-k)!}\) reprezinta nr de submultimi neordonate de cate k elemente ce se pot forma dintr-o multime cu n elemente.
    $$C_n^0 + C_n^1 + C_n^2 + ... + C_n^n = 2^n$$

    Numarul tuturor sumultimilor unei multimi cu n elemente este \(2^n\)

    Numarul submultimilor cu cate k elemente ale unei multimi cu n elemente este \(C_n^k\)

    Functia de gradul 2

    Varful parabolei este: \(V(\cfrac{-b}{2a} , \cfrac{-Δ}{4a})\)
    - daca a > 0 varful este punct de minim:
    -\(\cfrac{Δ}{4a}\) este valoare minima iar \(\cfrac{-b}{2a}\) este punct de minim
    - daca a < 0 este punct de maxim:
    -\(\cfrac{Δ}{4a}\) este valoare maxima iar \(\cfrac{-b}{2a}\) este punct de maxim

    Graficul functiei de gradul doi e tangent la axa Ox daca are Δ = 0

    Graficul functiei de gradul doi e situat deasupra axei Ox daca are:
    Δ < 0
    a > 0

    Relatiile lui Viette
    Pentru ecuatia de gradul doi cu radacini \(x_1 , x_2\) , au loc relatiile:
    \(x_1 + x_2 = \frac{-b}{a}\)
    \(x_1 · x_2 = \frac{c}{a}\)

    Observatie: \(x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2x_1x_2 = (\frac{-b}{a})^2 - \frac{2c}{a}\)

    Ecuatia cu radacini \(x_1 , x_2\) este \(x^2 - Sx + P = 0\) unde \(S = x_1 + x_2\) iar \(P = x_1 · x_2\)

    Conditia ca \(a^2 + bx + c >= 0\) , x din R este Δ <= 0 , a > 0
    Conditia ca \(a^2 + bx + c <= 0\) , x din R este Δ <= 0 , a < 0
    Conditia ca \(a^2 + bx + c > 0\) , x din R este Δ < 0 , a > 0
    Conditia ca \(a^2 + bx + c < 0\) , x din R este Δ < 0 , a < 0

    Conditia ca ecuatia \(a^2 + bx + c = 0\) sa aibe doua solutii reale este Δ > 0
    Conditia ca ecuatia \(a^2 + bx + c = 0\) sa aibe doua solutii egale este Δ = 0
    Conditia ca ecuatia \(a^2 + bx + c = 0\) sa nu aibe solutii reale este Δ < 0

    Functii

    Punctul \(A(a , b)\) se afla pe graficul functiei daca \(f(a) = b\)

    Pentru a afla punctele de intersectie dintre graficele a doua functii f si g se rezolva sistemul:
    $$\begin{cases} y = f(x) \\ y = g(x) \end{cases}$$ Solutiile (x , y) reprezinta coordonatele punctelor de intersectie.

    Inversa functiei f:
    Daca \(f(x) = y\) atunci \(f^{-1}(y) = x\)

    Intersectia cu Ox a graficului functiei f:
    Se rezolva ecuatia \(f(x) = 0\)
    Daca x este o solutie a ecuatiei \(f(x) = 0\) atunci punctul \(A(x , 0)\) este un punct de intersectie dintre axa Ox si graficul functiei f.

    Intersectia cu Oy a graficului functiei:
    Se calculeaza \(f(0)\) daca 0 e in domeniul de definitie.
    Punctul \(B(0 , f(0))\) reprezinta intersectia dintre axa Oy si graficul functiei f.
    In cazul in care 0 nu se afla in domeniul de definitie al functiei , graficul functiei nu taie axa Oy.

    Geometrie

    Ecuatia dreptei AB : \(\begin{vmatrix} x & y & 1 \\ x_A & y_A & 1 \\ x_B & y_B & 1 \end{vmatrix}\)

    Panta dreptei AB:
    daca stiu doua puncte panta este \(m_{AB} = (y_A - y_B)/(x_A - x_B)\)
    daca dreapta e data sub forma \(y = mx + n\) atunci m este panta
    daca ecuatia e sub forma \(ax + by + c = 0\) panta este \(\cfrac{-b}{a}\)
    OBS: dreptele verticale (\(x = a\)) nu eu panta

    Ecuatia unei drepte cand stiu un punct A si panta m este : \(y - y_A = m(x - x_A)\)

    Conditia de paralelism a doua drepte: \(d_1 || d_2 <=> m_{d_1} = m_{d_2}\)

    Distanta dintre doua puncte \(AB = \sqrt{(x_A - x_B)^2 + (y_A - y_B)^2}\)

    Mijlocul segmentului \(AB\) este \(M(\cfrac{x_A + x_B}{2} , \cfrac{y_A + y_B}{2})\)

    Conditia ca trei puncte A,B,C sa fie coliniare: \(\begin{vmatrix} x_A & y_A & 1 \\ x_B & y_B & 1 \\ x_C & y_C & 1 \end{vmatrix} = 0\)

    Punctul de intersectie dintre doua drepte se determina rezolvand sistemul facut de ecuatiile lor.

    Aria triunghiului ABC este \(S_{ABC} = \cfrac{Δ}{2}\) unde \(Δ = \begin{vmatrix} x_A & y_A & 1 \\ x_B & y_B & 1 \\ x_C & y_C & 1 \end{vmatrix}\)

    Aria triunghiului \(S_{ABC} = \cfrac{baza · inaltimea}{2}\)

    Aria triunghiului echilateral cu latura l este: \(S = \cfrac{l^3\sqrt3}{4}\)

    In triunghiul dreptunghic mediana e jumatate din ipotenuza

    Aria triunghiului ABC (Heron) \(S_{ABC} = \sqrt{p(p-a)(p-b)(p-c)}\) unde \(p = \cfrac{a + b + c}{2}\)

    Aria triunghiului \(ABC = \cfrac{BC · AC · \sin C}{2} = \cfrac{BC · AB · \sin B}{2} = \cfrac{AB · AC · \sin A}{2}\)

    Teorema lui Pitagora in triunghiul dreptunghic \(b^2 + c^2 = a^2\)

    Teorema cosinusului
    \(BC^2 = AC^2 + AB^2 - 2·AB·AC·\cos A\)

    \(AC^2 = AB^2 + BC^2 - 2·AB·BC·\cos B\)

    \(AB^2 = AC^2 + BC^2 - 2·BC·AC·\cos C \)

    Teorema sinusurilor \(\cfrac{BC}{\sin A} = \cfrac{AC}{\sin B} = \cfrac{AB}{\sin C} = 2R\) unde R raza cercului circumscris triunghiului

    Prima bisectoarea este bisectoarea cadranului 1 in reperul xOy si are ecuatia \(y = x\).

    A doua bisectoarea este bisectoarea cadranului 2 in reperul xOy si are ecuatia \(y=-x\).

    Mediana in trunghi este segmentul ce uneste un varf cu mijlocul laturii opuse

    Mediatoarea unui segment e perpendiculara pe mijlocul segmentului

    Inaltimea in triunghi e perpendiculara din varf pe latura opusa

    Bisectoarea este semidreapta care imparte un unghi in 2 unghiuri congruente.

    In trunghiul dreptunghic \(\sin = \cfrac{Cateta Opusa}{ipotenuza}\) , \(\cos = \cfrac{Cateta Alaturata}{ipotenuza}\)

    Vectori in plan

    Modulul vectorului: \(|\overrightarrow{v}| = \sqrt{a^2 + b^2}\)

    Produsul scalar a doi vectori \(\overrightarrow v = a · \overrightarrow i + b · \overrightarrow j\) si \(\overrightarrow w = c · \overrightarrow i + d · \overrightarrow j\) este \(\overrightarrow v + \overrightarrow w = a · c + b · d\)

    Suma a doi vectori \(\overrightarrow v = a · \overrightarrow i + b · \overrightarrow j\) si \(\overrightarrow w = c · \overrightarrow i + d · \overrightarrow j\) este \(\overrightarrow v + \overrightarrow w = (a + c)\overrightarrow i + (b + d)\overrightarrow j\)

    Conditia ca doi vectori sa fie coliniari :
    Doi vectori \(\overrightarrow v\) si \(\overrightarrow u\) sunt coliniari daca exista a numar real astfel incat \(\overrightarrow v = a · \overrightarrow u\)
    Daca vectorii sunt dati sub forma \(\overrightarrow{v} = a · \overrightarrow i + b · \overrightarrow j\) si \(\overrightarrow u = c · \overrightarrow i + d · \overrightarrow j\) conditia de coliniaritate revine la \(\cfrac{a}{c} = \cfrac{b}{d}\)

    Daca \(A(x_A , y_A)\) si \(B(x_B , y_B)\) atunci \(\overrightarrow{AB} = (x_B - x_A) · \overrightarrow i + (y_B - y_A) · \overrightarrow j\)

    Daca \(A(x_A , y_A)\) vectorul de pozitie al lui A este \(\overrightarrow{OA} = x_A · \overrightarrow i + y_A · \overrightarrow j\) se mai noteaza \(\overrightarrow r_A\)

    Logaritmi

    \(\log_a b\) = puterea la care il ridic pe a astfel incat sa dea pe b.

    \(\log_a b\) exista doar pentru \(a>0 , b>0 , a \not = 1\)

    \(\log_a {a^b} = b\) (cu ajutorul acestei formule orice numar real poate fi scris ca \(\log\) in orice baza vreau)

    \(\log_a b = c\) revine la \(b = a^c\)

    \(\log_a b + \log_a^c = \log_a(bc)\)

    \(\log_a b - \log_a^c = \log_a(b/c)\)

    \(\log_a {b^p} = p \log_a b\)

    \(\log_a {p^b} = 1/p \log_a b\)

    \(\log_a b = 1/\log_b a\)

    \(a^{\log_a b} = b\)

    daca \(a > 1\) functia \(\log\) este crescatoare , \(\log_a b > \log_a c => b > c\)

    daca \(a < 1\) functia \(\log\) este crescatoare , \(\log_a b > \log_a c => b < c\)
