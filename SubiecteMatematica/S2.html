    Algebra

    Functia de gradul 2

    Varful parabolei este: V(-b/2a , -Δ/4a)
    - daca a > 0 varful este punct de minim:
    -Δ/4a este valoare minima iar -b/2a este punct de minim
    - daca a < 0 este punct de maxim:
    -Δ/4a este valoare maxima iar -b/2a este punct de maxim

    Graficul functiei de gradul doi e tangent la axa Ox daca are Δ = 0

    Graficul functiei de gradul doi e situat deasupra axei Ox daca are:
    Δ < 0
    a > 0

    Relatiile lui Viette
    Pentru ecuatia de gradul doi cu radacini x1 , x2 , au loc relatiile:
    x1 + x2 = -b/a
    x1 · x2 = c/a

    Observatie: x12 + x22 = (x1 + x2)2 - 2x1x2 = (-b/a)2 - 2c/a

    Ecuatia cu radacini x1 , x2 este x2 - Sx + P = 0 unde S = x1 + x2 iar P = x1 · x2

    Conditia ca a2 + bx + c >= 0 , x din R este Δ <= 0 , a > 0
    Conditia ca a2 + bx + c <= 0 , x din R este Δ <= 0 , a < 0
    Conditia ca a2 + bx + c > 0 , x din R este Δ < 0 , a > 0
    Conditia ca a2 + bx + c < 0 , x din R este Δ < 0 , a < 0

    Conditia ca ecuatia a2 + bx + c = 0 sa aibe doua solutii reale este Δ > 0
    Conditia ca ecuatia a2 + bx + c = 0 sa aibe doua solutii egale este Δ = 0
    Conditia ca ecuatia a2 + bx + c = 0 sa nu aibe solutii reale este Δ < 0

    Ecuatia de gradul 2

    Ecuatia ax2 + bx + c = 0. Se calculeaza Δ = b2 - 4ac
    Daca Δ > 0 atunci ecuatia de gradul 2 are doua radacini reale diferite date de formula x1,2 = (-b ± √Δ)/2a
    Daca Δ = 0 atunci ecuatia de gradul 2 are doua radacini reale egale date de formula x1,2 = -b/2a
    Daca Δ < 0 atunci ecuatia de gradul 2 are doua radacini complexe diferite date de formula x1,2 = (-b ± i√-Δ)/2a

    ax2 + bx + c = a(x - x1)(x - x2)

    Progresii aritmetice

    Definitie: Se numeste progresie aritmetica un sir de numere reale in care fiecare termen se obtine din termenul anterior adunand o constanta numita ratie (r)

    Proprietate: Trei numere a , b, c sunt in progresie aritmetica daca b e medie aritmetica intre a si c adica b = (a + c)/2

    an=a1+(n - 1)r

    Sn=(n(2a1 + (n - 1)r))/2 unde am notat Sn = a1 + a2 + a3 + ... + an

    Progresii geometrice

    Definitie: Se numeste progresie geometrica un sir de numere reale in care fiecare termen se obtine din termenul anterior inmultind cu o constanta numita ratie (q).

    Proprietate: Trei numere a , b, c sunt in progresie geometrica cu termeni pozitivi daca b e medie geometrica intre a si c adica b = √(ac)

    an=a1qn - 1

    Sn = a1(qn-1)/(q - 1) unde am notat Sn = a1 + a2 + a3 + ... + an

    Elemente de combinatorica

    Permutari de n se noteaza Pn
    Pn=n! si reprezinta numarul de multimi ordonate ce se pot forma cu n elemente.

    Aranjamente de n luate cate k se noteaza Ank
    Ank = n!/(n-k)! reprezinta numarul de submultimi ordonate de cate k elemente ce se pot forma dintr-o multime de n elemente.

    Combinari de n luate cate k se noteaza Cnk
    Cnk = n!/k!(n-k)! reprezinta nr de submultimi neordonate de cate k elemente ce se pot forma dintr-o multime cu n elemente.
    Cn0 + Cn1 + Cn2 + ... + Cnn = 2n

    Numarul tuturor sumultimilor unei multimi cu n elemente este 2n

    Numarul submultimilor cu cate k elemente ale unei multimi cu n elemente este Cnk

    Binomul lui Newton:
    (a + b)n = Cn0an + Cn1an-1b + Cn2an-2b2 + ... + Cnkan - kbk + ... + Cnnbn

    Formula termenului general din binomul lui Newton este :
    Tk+1 = Cnkan-kbk

    Logaritmi

    logab = puterea la care il ridic pe a astfel incat sa dea pe b.

    logab exista doar pentru a>0 , b>0 , a != 1

    logaab = b (cu ajutorul acestei formule orice numar real poate fi scris ca log in orice baza vreau)

    logab = c revine la b = ac

    logab + logac = loga(bc)

    logab - logac = loga(b/c)

    logabp = p logab

    logapb = 1/p logab

    logab = 1/logba

    alogab = b

    daca a > 1 functia log este crescatoare , logab > logac => b > c

    daca a < 1 functia log este crescatoare , logab > logac => b < c

    Legi de compozitie

    Fie M o multime nevida pe care s-a dat o lege de compozitie notata *.
    Legea * este asociativa daca (x * y) * z = x * (y * z) ∀x , y , z ∈ M
    Legea * este comutativa daca x * y = y * x ∀x , y ∈ M
    Legea * are element neutru edaca x * e = e * x = x ∀x ∈ M
    Un element x ∈ M se numeste simetrizabil daca ∃ x' ∈ M astfel incat x * x' = x' * x = e

    Asimptote

    Asimptote orizontale

    Pentru a studia existenţa asimptotei orizontale spre +∞ la graficul unei funcţii se calculează limx->+∞f(x).

    Cazul 1. Daca aceasta limita nu exista sau este infinita atunci graficul nu are asimptota orizontala spre +∞
    Cazul 2. Daca aceasta limita exista si este finita, egala cu un numar real l , atunci graficul are asimptota orizontala spre +∞ dreapta de ecuatie y = l
    Analog se studiaza existenta asimptotei orizontale spre -∞

    Asimptote oblice

    Asimptota oblica spre +∞ (daca exista) are ecuatia y = mx + n unde m si n se calculeaza cu formulele:
    m = limx->+∞f(x)/x
    n = limx->+∞[f(x) - m • x]
    Analog se studiaza existenta asimptotei oblice spre -∞

    Asimptote verticale

    Se calculeaza limx->x0 x<x0f(x) si limx->x0 x>x0f(x)
    Daca una dintre aceste limite este infinita atunci graficul are asimptota verticala dreapta de ecuatie x = x0

    Derivate

    Derivata unei functii intr-un punct: f'(x0) = limx->x0(f(x) - f(x0))/(x - x0)

    Tangenta la graficul unei functii in punctul de abscisa x0:
    y - f(x0) = f'(x0)(x - x0)

    Reguli de derivare

    (f + g)' = f' + g'
    (f - g)' = f' - g'
    (c • f)' = c • f'
    (f • g)' = f' • g + f • g'
    (f/g)' = (f' • g - f • g')/g2

    Derivatele unor functii uzuale

    c' = 0 (lnx)' = 1/x
    x' = 1 (logax)' = 1/x • lna
    (xn)' = nxn-1 (sinx)' = cosx
    (1/x)' = -1/x2 (cosx)' = -sinx
    (√x)' = 1/2√x (tgx)' = 1/cos2x
    (ex)' = ex (ctgx)' = -1/sin2x
    (e-x)' = -e-x (arcsinx)' = 1/√(1 - x2)
    ax = ax • lna (arccosx)' = -1/√(1 - x2)
    (arctgx)' = 1/(1 + x2) (arcctgx)' = -1/(1 + x2)

    Derivatele unor functii compuse

    (un)' = n • un-1 • u' (lnu)' = u'/u
    (1/u)' = -u2/u (logau)' = u'/u • lna
    (√u)' = u'/2√u (sinu)' = cosu • u'
    (eu)' = eu • u' (cosu)' = -sinu • u'
    (e-u)' = -e-u • u' (tgu)' = u'/cons2u
    (au)' = au • lna • u' (ctgu)' = -u'/sin2u
    (arcsinu)' = u'/√(1-u2) (arccosu)' = -u'/√(1 - u2)
    (arctgu)' = u'/(1 + u2) (arcctgu)' = -u'/(1 + u2)

    Integrale

    Integrale nedefinite

    ∫1 dx = x + C ∫sinx dx = -cosx + C
    ∫x dx = x2/2 + C ∫cox dx = sinx + C
    ∫xn dx = xn + 1/(n + 1) + C ∫1/cos2x dx = tgx + C
    ∫1/x dx = ln|x| + C ∫1/sin2x dx = -ctgx + C
    ∫ex dx = ex + C ∫1/(x2 + a2) dx = 1/a • arctg(x/a) + C
    ∫e-x = -e-x + C ∫1/(x2 - a2) dx = 1/2a • ln|(x - a)/(x + a)| + C
    ∫ax dx = ax/lna + C ∫1/√(x2 + a2) dx = ln(x + √(x2 + a2)) + C
    ∫1/√(x2 - a2) dx = ln|x + √(x2 - a2)| + C ∫1/√(a2 - x2) dx = arcsin(x/a) + C
    ∫u'(x) dx = u(x) + C ∫sinu(x) • u'(x) dx = -cosu(x) + C
    ∫un(x) • u'(x) dx = un + 1(x)/(n + 1) + C ∫cosu(x) • u'(x) dx = sinu(x) + C
    ∫u'(x)/u(x) dx = ln|u(x)| + C ∫u'(x)/cos2u(x) dx = tgu(x) + C
    ∫eu(x)u'(x) dx = eu(x) + C ∫u'(x)/sin2u(x) dx = -ctgu(x) + C
    ∫e-u(x)u'(x) dx = -e-u(x) + C ∫u'(x)/(u2(x) + a2) dx = 1/a • arctg(u(x)/a) + C
    ∫au(x)u'(x) dx = au(x)/lna + C ∫u'(x)/(u2 - a2) dx = 1/2a • ln|(u(x) - a)/(u(x) + a)| + C
    ∫u'(x)/√(u2 + a2) dx = ln(u(x) + √(u2(x) + a2)) + C ∫u'(x)/√(u2 - a2) dx = ln|u(x) + √(u2(x) - a2)| + C
    ∫u'(x)/√(a2 - u2(x)) dx = arcsin(u(x)/a) + C

    Formula de integrare prin parti pentru integrale nedefinite este:
    ∫f(x)g'(x)dx = f(x)g(x) - ∫f'(x)g(x)

    Formula de integrare prin parti pentru integrale nedefinite este:
    ∫abf(x)g'(x)dx = f(x)g(x)|ab - ∫abf'(x)g(x)

    Geometrie

    Teorema lui Pitagora

    Intr-un triunghi dreptunghic are loc relatia:
    cateta2 + cateta2 = ipotenuza2

    Teorema lui Pitagora generalizata(teorema cosinusului)

    Intr-un triunghi oarecare ABC are loc relatia:
    BC2 = AB2 + AC2 - 2 • AB • AC • cosA

    Aria unui triunghi echilateral de latura l este:

    Aria = l2√3/4

    Aria unui triunghi oarecare

    Aria = AB • AC • sinA/2

    Formula lui Heron

    S = √(p(p-a)(p-b)(p-c)) , p = (a + b + c)/2

    Aria triunghiului dreptunghic

    Aria = cateta • cateta / 2

    Teoria sinusurilor

    Intr-un triunghi oarecare ABC are loc relatia:
    a/sinA = b/sinB = c/sinC
    unde a , b , c sunt laturile triunghiului
    A , B , C sunt unghiurile triunghiului
    R este raza cercului circumscris triunghiului

    Distanta dintre doua puncte

    Daca A(x1 , y1) si B(x2 , y2) sunt doua puncte in plan atunci mijlocul segmentului AB este
    M((x1 + x2)/2 , (y1 + y2)/2)

    Vectorul de pozitie al unui punct

    Daca A(x , y) atunci OA⃗ = x • i⃗ + y • j⃗
    Daca A(x1 , y1) si B(x2 , y2) sunt doua puncte in plan atunci vector AB⃗ este dat de formula:
    AB⃗ = (x2 - x1)i⃗ + (y2 - y1)j⃗

    Ecuatia unei drepte care trece prin doua puncte date

    Daca A(x1 , y1) si B(x2 , y2) sunt doua puncte in plan atunci ecuatia dreptei AB se poate afla cu formula:
    (x - x1)/(x2 - x1) = (y - y1)/(y2 - y1) sau cu formula:

    x y 1
    x1 y1 1
    x2 y2 1
    = 0

    Ecuatia unei drepte care trece prin punctul A(x0 , y0) si are panta data m

    Este data de formula y - y0 = m(x - x0)

    Conditia de coliniaritate a trei puncte in plan

    Fie A(x1 , y1) , B(x2 , y2) , C(x3 , y3) trei puncte in plan.
    Punctele A , B , C sunt coliniare daca si numai daca

    x1 y1 1
    x2 y2 1
    x3 y3 1
    = 0

    Aria unui triunghi

    Fie A(x1 , y1) , B(x2 , y2) , C(x3 , y3) trei puncte in plan.
    Aria triunghiului ABC este data de formula
    A△ABC = 1/x • |Δ|
    unde Δ este urmatorul determinant:

    x1 y1 1
    x2 y2 1
    x3 y3 1

    Distanta de la un punct la o dreapta

    Daca A(x0 , y0) este un punct si d: ax + by + x = 0 este o dreapta in plan atunci distanta de la punctul A la dreapta d este data de formula:
    dist(A , d) = |ax0 + by0 + c|/√(a2 + b2)

    Panta unei drepte

    Daca A(x1 , y1) si B(x2 , y2) sunt doua puncte in plan atunci panta dreptei AB este data de formula:
    m = (y2 - y1)/(x2 - x1)

    Conditia de coliniaritate a doi vectori in plan

    Fie v⃗ = a1i⃗ + b1j⃗ si v2 = a2i⃗ + b2j⃗ doi vectori in plan. Conditia de coliniaritate a vectorilor v1⃗ , v2⃗ este:
    a1/a2 = b1/b2

    Conditia de perpendicularitate a doi vectori in plan

    Fie v⃗ = a1i⃗ + b1j⃗ si v2 = a2i⃗ + b2j⃗ doi vectori in plan. Avem:
    v1⃗ ⊥ v2⃗ <=> a1 • a2 + b1 • b2 = 0 (produsul scalar este 0)

    Conditia de paralelism a doua drepte in plan

    Doua d1 si d2 sunt paralele daca si numai daca au aceeasi panta adica:
    d1 || d2 <=> md1 = md2
    Altfel, daca dreptele sunt date prin ecuatia generala: d1: a1x + b1y + c1 = 0 si d2: a2x + b2y + c = 0
    atunci dreptele sunt paralele daca a1/a2 = b1/b2

    Conditia de perpendicularitate a doua drepte in plan

    Doua drepte d1 si d2 sunt perpendiculare daca si numai daca produsul pantelor este egal cu -1 adica:
    d1 ⊥ d2 <=> md1 • md1 = -1
