    Algebra

    Functii

    Punctul \(A(a , b)\) se afla pe graficul functiei daca \(f(a) = b\)

    Pentru a afla punctele de intersectie dintre graficele a doua functii f si g se rezolva sistemul:
    $$\begin{cases} y = f(x) \\ y = g(x) \end{cases}$$ Solutiile (x , y) reprezinta coordonatele punctelor de intersectie.

    Inversa functiei f:
    Daca \(f(x) = y\) atunci \(f^{-1}(y) = x\)

    Intersectia cu Ox a graficului functiei f:
    Se rezolva ecuatia \(f(x) = 0\)
    Daca x este o solutie a ecuatiei \(f(x) = 0\) atunci punctul \(A(x , 0)\) este un punct de intersectie dintre axa Ox si graficul functiei f.

    Intersectia cu Oy a graficului functiei:
    Se calculeaza \(f(0)\) daca 0 e in domeniul de definitie.
    Punctul \(B(0 , f(0))\) reprezinta intersectia dintre axa Oy si graficul functiei f.
    In cazul in care 0 nu se afla in domeniul de definitie al functiei , graficul functiei nu taie axa Oy.

    Elemente de combinatorica

    Permutari de n se noteaza \(P_n\)
    \(P_n=n!\) si reprezinta numarul de multimi ordonate ce se pot forma cu n elemente.

    Aranjamente de n luate cate k se noteaza \(A_n^k\)
    \(A_n^k = \cfrac{n!}{(n-k)!}\) reprezinta numarul de submultimi ordonate de cate k elemente ce se pot forma dintr-o multime de n elemente.

    Combinari de n luate cate k se noteaza \(C_n^k\)
    \(C_n^k = \cfrac{n!}{k!(n-k)!}\) reprezinta nr de submultimi neordonate de cate k elemente ce se pot forma dintr-o multime cu n elemente.
    $$C_n^0 + C_n^1 + C_n^2 + ... + C_n^n = 2^n$$

    Numarul tuturor sumultimilor unei multimi cu n elemente este \(2^n\)

    Numarul submultimilor cu cate k elemente ale unei multimi cu n elemente este \(C_n^k\)

    Binomul lui Newton:
    $$(a + b)^{n} = C_{n}^{0}a^{n} + C_{n}^{1}a^{n-1}b + C_{n}^{2}a^{n-2}b^{2} + ... + C_{n}^{k}a^{n - k}b^{k} + ... + C_{n}^{n}b^{n}$$

    Formula termenului general din binomul lui Newton este :
    $$T_{k+1} = C_{n}^{k}a^{n-k}b^{k}$$

    Logaritmi

    \(\log_a b\) = puterea la care il ridic pe a astfel incat sa dea pe b.

    \(\log_a b\) exista doar pentru \(a>0 , b>0 , a \not = 1\)

    \(\log_a {a^b} = b\) (cu ajutorul acestei formule orice numar real poate fi scris ca \(\log\) in orice baza vreau)

    \(\log_a b = c\) revine la \(b = a^c\)

    \(\log_a b + \log_a^c = \log_a(bc)\)

    \(\log_a b - \log_a^c = \log_a(b/c)\)

    \(\log_a {b^p} = p \log_a b\)

    \(\log_a {p^b} = 1/p \log_a b\)

    \(\log_a b = 1/\log_b a\)

    \(a^{\log_a b} = b\)

    daca \(a > 1\) functia \(\log\) este crescatoare , \(\log_a b > \log_a c => b > c\)

    daca \(a < 1\) functia \(\log\) este crescatoare , \(\log_a b > \log_a c => b < c\)

    Legi de compozitie

    Fie M o multime nevida pe care s-a dat o lege de compozitie notata *.
    Legea * este asociativa daca \((x * y) * z = x * (y * z)\) \(∀x , y , z ∈ M\)
    Legea * este comutativa daca \(x * y = y * x\) \(∀x , y ∈ M\)
    Legea * are element neutru edaca \(x * e = e * x = x\) \(∀x ∈ M\)
    Un element \(x ∈ M\) se numeste simetrizabil daca \(∃ x' ∈ M\) astfel incat \(x * x' = x' * x = e\)

    Asimptote

    Asimptote orizontale

    Pentru a studia existenţa asimptotei orizontale spre \(+∞\) la graficul unei funcţii se calculează \(\lim_{x->+∞} f(x)\).

    Cazul 1. Daca aceasta limita nu exista sau este infinita atunci graficul nu are asimptota orizontala spre \(+∞\)
    Cazul 2. Daca aceasta limita exista si este finita, egala cu un numar real \(l\) , atunci graficul are asimptota orizontala spre \(+∞\) dreapta de ecuatie \(y = l\)
    Analog se studiaza existenta asimptotei orizontale spre \(-∞\)

    Asimptote oblice

    Asimptota oblica spre \(+∞\) (daca exista) are ecuatia \(y = mx + n\) unde m si n se calculeaza cu formulele:
    \(m = lim_{x->+∞}\cfrac{f(x)}{x}\)
    \(n = lim_{x->+∞}[f(x) - m \cdot x]\)
    Analog se studiaza existenta asimptotei oblice spre \(-∞\)

    Asimptote verticale

    Se calculeaza \(lim_{\begin{cases}x->x_0 \\ x<x_0\end{cases}}f(x)\) si \(lim_{\begin{cases}x->x_0 \\ x>x_0\end{cases}}f(x)\)
    Daca una dintre aceste limite este infinita atunci graficul are asimptota verticala dreapta de ecuatie \(x = x_{0}\)

    Derivata unei functii intr-un punct: \(f'(x_{0}) = \cfrac{lim_{x->x_0}(f(x) - f(x_{0}))}{x - x_{0}}\)

    \(y - f(x_{0}) = f'(x_{0})(x - x_{0})\)

    Reguli de derivare

    \((f + g)' = f' + g'\)

    \((f - g)' = f' - g'\)

    \((c \cdot f)' = c \cdot f'\)

    \((f \cdot g)' = f' \cdot g + f \cdot g'\)

    \((f/g)' = (f' \cdot g - f \cdot g')/g^{2}\)

    Derivatele unor functii uzuale

    \(c' = 0\) \((lnx)' = \cfrac{1}x\)
    \(x' = 1\) \((log_{a}x)' = \cfrac{1}x \cdot lna\)
    \((x^{n})' = nx^{n-1}\) \((\sin x)' = \cos x\)
    \((\cfrac{1}x)' = \cfrac{-1}{x^{2}}\) \((\cos x)' = -\sin x\)
    \((\sqrt{x})' = \cfrac{1}{2\sqrt{x}}\) \((tg x)' = \cfrac{1}{\cos^{2}x}\)
    \((e^{x})' = e^{x}\) \((ctgx)' = \cfrac{-1}{\sin^{2}x}\)
    \((e^{-x})' = -e^{-x}\) \((arcsin x)' = \cfrac{1}{\sqrt{1 - x^{2}}}\)
    \(a^{x} = a^{x} \cdot lna\) \((arccos x)' = \cfrac{-1}{\sqrt{1 - x^{2}}}\)
    \((arctg x)' = \cfrac{1}{1 + x^{2}}\) \((arcctg x)' = \cfrac{-1}{1 + x^{2}}\)

    Derivatele unor functii compuse

    \((u^{n})' = n \cdot u^{n-1} \cdot u'\) \((ln u)' = \cfrac{u'}{u}\)

    \((arcsin u)' = \cfrac{u'}{\sqrt{1-u^{2}}}\) \((arccos u)' = \cfrac{-u'}{\sqrt{1 - u^{2}}}\)
    \((arctg u)' = \cfrac{u'}{(1 + u^{2})}\) \((arcctg u)' = \cfrac{-u'}{1 + u^{2}}\)

    Integrale

    Integrale nedefinite

    \(\int 1 dx = x + C\) \(\int \sin x dx = -\cos x + C\)
    \(\int x dx = \cfrac{x^{2}}{2} + C\) \(\int cox dx = \sin x + C\)
    \(\int x^{n} dx = \cfrac{x^{n + 1}}{n + 1} + C\) \(\int \cfrac{1}{\cos^{2}x}dx = tgx + C\)
    \(\int \cfrac{1}{x} dx = ln|x| + C\) \(\int \cfrac{1}{\sin^{2}x} dx = -ctgx + C\)
    \(\int e^{x} dx = e^{x} + C\) \(\int \cfrac{1}{x^{2} + a^{2}} dx = 1/a \cdot arctg(\cfrac{x}{a}) + C\)
    \(\int e^{-x} = -e^{-x} + C\) \(\int \cfrac{1}{x^{2} - a^{2}} dx = \cfrac{1}{2a} \cdot ln|\cfrac{x - a}{x + a}| + C\)
    \(\int a^{x} dx = \cfrac{a^{x}}{ln a} + C\) \(\int \cfrac{1}{\sqrt{x^{2} + a^{2}}} dx = ln(x + \sqrt{x^{2} + a^{2}}) + C\)
    \(\int \cfrac{1}{\sqrt{x^{2} - a^{2}}} dx = ln|x + \sqrt{x^{2} - a^{2}}| + C\) \(\int \cfrac{1}{\sqrt{a^{2} - x^{2}}} dx = arcsin(\cfrac{x}{a}) + C\)
    \(\int u'(x) dx = u(x) + C\) \(\int \sin u(x) \cdot u'(x) dx = -\cos u(x) + C\)
    \(\int u^{n}(x) \cdot u'(x) dx = \cfrac{u^{n + 1}(x)}{n + 1} + C\) \(\int \cos u(x) \cdot u'(x) dx = \sin u(x) + C\)
    \(\int \cfrac{u'(x)}{u(x)} dx = ln|u(x)| + C\) \(\int \cfrac{u'(x)}{\cos^{2}u(x)} dx = tg u(x) + C\)
    \(\int e^{u(x)}u'(x) dx = e^{u(x)} + C\) \(\int \cfrac{u'(x)}{\sin^{2}u(x)} dx = -ctgu(x) + C\)
    \(\int e^{-u(x)}u'(x) dx = -e^{-u(x)} + C\) \(\int \cfrac{u'(x)}{u^{2}(x) + a^{2}} dx = \cfrac{1}{a} \cdot arctg(\cfrac{u(x)}{a}) + C\)
    \(\int a^{u(x)}u'(x) dx = \cfrac{a^{u(x)}}{lna} + C\) \(\int \cfrac{u'(x)}{u^{2} - a^{2}} dx = \cfrac{1}{2a} \cdot ln|\cfrac{u(x) - a}{u(x) + a}| + C\)
    \(\int \cfrac{u'(x)}{\sqrt{u^{2} + a^{2}}} dx = ln(u(x) + \sqrt{u^{2}(x) + a^{2}} + C\) \(\int \cfrac{u'(x)}{\sqrt{u^{2} - a^{2}}} dx = ln|u(x) + \sqrt{u^{2}(x) - a^{2}}| + C\)
    \(\int \cfrac{u'(x)}{\sqrt{a^{2} - u^{2}(x)}} dx = arcsin(\cfrac{u(x)}{a}) + C\)

    Formula de integrare prin parti pentru integrale nedefinite este:
    $$\int f(x)g'(x)dx = f(x)g(x) - \int f'(x)g(x)$$

    Formula de integrare prin parti pentru integrale nedefinite este:
    $$\int_{a}^{b}f(x)g'(x)dx = f(x)g(x)|_{a}^{b} - \int_{a}^{b}f'(x)g(x)$$

    \(BC^{2} = AB^{2} + AC^{2} - 2 \cdot AB \cdot AC \cdot \cos A\)

    Aria unui triunghi echilateral de latura l este:

    \(Aria = l^{2}\sqrt{\cfrac{3}{4}}\)

    Aria unui triunghi oarecare

    \(Aria = \cfrac{AB \cdot AC \cdot \sin A}{2}\)

    Formula lui Heron

    \(S = \sqrt{p(p-a)(p-b)(p-c)} , p = \cfrac{a + b + c}{2}\)

    Aria triunghiului dreptunghic

    \(Aria = \cfrac{cateta1 \cdot cateta2}{2}\)

    Teoria sinusurilor

    Intr-un triunghi oarecare ABC are loc relatia:
    \(\cfrac{a}{\sin A} = \cfrac{b}{\sin B} = \cfrac{c}{\sin C}\)
    unde \(a , b , c\) sunt laturile triunghiului
    \(A , B , C\) sunt unghiurile triunghiului
    \(R\) este raza cercului circumscris triunghiului

    Distanta dintre doua puncte

    \(Daca A(x_{1} , y_{1})\) si \(B(x_{2} , y_{2})\) sunt doua puncte in plan atunci mijlocul segmentului \(AB\) este
    \(M((x_{1} + x_{2})/2 , (y_{1} + y_{2})/2)\)

    Vectorul de pozitie al unui punct

    Daca \(A(x , y)\) atunci \(\overrightarrow{OA} = x \cdot \overrightarrow i + y \cdot \overrightarrow j\)
    Daca \(A(x_{1} , y_{1})\) si \(B(x_{2} , y_{2})\) sunt doua puncte in plan atunci vector \(\overrightarrow{AB}\) este dat de formula:
    \(\overrightarrow{AB} = (x_{2} - x_{1})\overrightarrow i + (y_{2} - y_{1})\overrightarrow j\)

    Ecuatia unei drepte care trece prin doua puncte date

    \(Daca A(x_{1} , y_{1}) si B(x_{2} , y_{2})\) sunt doua puncte in plan atunci ecuatia dreptei \(AB\) se poate afla cu formula:
    \(\cfrac{x - x_{1}}{x_{2} - x_{1}} = \cfrac{y - y_{1}}{y_{2} - y_{1}}\) sau cu formula:
    $$\begin{vmatrix}x & y & 1 \\ x_{1} & y_{1} & 1 \\ x_{2} & y_{2} & 1\end{vmatrix} = 0$$

    Ecuatia unei drepte care trece prin punctul \(A(x_{0} , y_{0})\) si are panta data \(m\)

    Este data de formula \(y - y_{0} = m(x - x_{0})\)

    Conditia de coliniaritate a trei puncte in plan

    Fie \(A(x_{1} , y_{1}) , B(x_{2} , y_{2}) , C(x_{3} , y_{3})\) trei puncte in plan.
    Punctele \(A , B , C\) sunt coliniare daca si numai daca
    $$\begin{vmatrix}x_{1} & y_{1} & 1 \\ x_{2} & y_{2} & 1 \\ x_{3} & y_{3} & 1\end{vmatrix} = 0$$

    Aria unui triunghi

    Fie \(A(x_{1} , y_{1}) , B(x_{2} , y_{2}) , C(x_{3} , y_{3})\) trei puncte in plan.
    Aria triunghiului ABC este data de formula
    \(A_{△ABC} = \cfrac{1}{x} \cdot |Δ|\)
    unde \(Δ\) este urmatorul determinant:
    $$\begin{vmatrix}x_{1} & y_{1} & 1 \\ x_{2} & y_{2} & 1 \\ x_{3} & y_{3} & 1\end{vmatrix}$$

    Distanta de la un punct la o dreapta

    Daca \(A(x_{0} , y_{0})\) este un punct si \(d: ax + by + x = 0\) este o dreapta in plan atunci distanta de la punctul \(A\) la dreapta \(d\) este data de formula:
    \(dist(A , d) = \cfrac{|ax_{0} + by_{0} + c|}{\sqrt{{a^{2} + b^{2}}}}\)

    Panta unei drepte

    Daca \(A(x_{1} , y_{1})\) si \(B(x_{2} , y_{2})\) sunt doua puncte in plan atunci panta dreptei \(AB\) este data de formula:
    $$m = \cfrac{y_{2} - y_{1}}{x_{2} - x_{1}}$$

    Conditia de coliniaritate a doi vectori in plan

    Fie \(\overrightarrow v = a_{1}\overrightarrow i + b_{1}\overrightarrow j\) si \(v_{2} = a_{2}\overrightarrow i + b_{2}\overrightarrow j\) doi vectori in plan. Conditia de coliniaritate a vectorilor \(\overrightarrow v_{1} , \overrightarrow v_{2}\) este:
    $$\cfrac{a_{1}}{a_{2}} = \cfrac{b_{1}}{b_{2}}$$

    Conditia de perpendicularitate a doi vectori in plan

    Fie \(\overrightarrow v = a_{1}\overrightarrow i + b_{1}\overrightarrow j\) si \(v_{2} = a_{2}\overrightarrow i + b_{2}\overrightarrow j\) doi vectori in plan. Avem:
    $$\overrightarrow v_{1} ⊥ \overrightarrow v_{2} <=> a_{1} \cdot a_{2} + b_{1} \cdot b_{2} = 0$$ (produsul scalar este 0)

    Conditia de paralelism a doua drepte in plan

    Doua \(d_{1}\) si \(d_{2}\) sunt paralele daca si numai daca au aceeasi panta adica:
    \(d_{1} || d_{2} <=> m_{d_{1}} = m_{d_{2}}\)
    Altfel, daca dreptele sunt date prin ecuatia generala: \(d_{1}: a_{1}x + b_{1}y + c_{1} = 0 si d_{2}: a_{2}x + b_{2}y + c = 0\)
    atunci dreptele sunt paralele daca \(\cfrac{a_{1}}{a_{2}} = \cfrac{b_{1}}{b_{2}}\)

    Conditia de perpendicularitate a doua drepte in plan

    Doua drepte \(d_{1}\) si \(d_{2}\) sunt perpendiculare daca si numai daca produsul pantelor este egal cu -1 adica:
    \(d_{1} ⊥ d_{2} <=> m_{d_{1}} \cdot m_{d_{1}} = -1\)

    Modulul vectorului: \(|\overrightarrow{v}| = \sqrt{a^2 + b^2}\)

    Produsul scalar a doi vectori \(\overrightarrow v = a · \overrightarrow i + b · \overrightarrow j\) si \(\overrightarrow w = c · \overrightarrow i + d · \overrightarrow j\) este \(\overrightarrow v + \overrightarrow w = a · c + b · d\)

    Suma a doi vectori \(\overrightarrow v = a · \overrightarrow i + b · \overrightarrow j\) si \(\overrightarrow w = c · \overrightarrow i + d · \overrightarrow j\) este \(\overrightarrow v + \overrightarrow w = (a + c)\overrightarrow i + (b + d)\overrightarrow j\)

    Conditia ca doi vectori sa fie coliniari :
    Doi vectori \(\overrightarrow v\) si \(\overrightarrow u\) sunt coliniari daca exista a numar real astfel incat \(\overrightarrow v = a · \overrightarrow u\)
    Daca vectorii sunt dati sub forma \(\overrightarrow{v} = a · \overrightarrow i + b · \overrightarrow j\) si \(\overrightarrow u = c · \overrightarrow i + d · \overrightarrow j\) conditia de coliniaritate revine la \(\cfrac{a}{c} = \cfrac{b}{d}\)

    Daca \(A(x_A , y_A)\) si \(B(x_B , y_B)\) atunci \(\overrightarrow{AB} = (x_B - x_A) · \overrightarrow i + (y_B - y_A) · \overrightarrow j\)

    Daca \(A(x_A , y_A)\) vectorul de pozitie al lui A este \(\overrightarrow{OA} = x_A · \overrightarrow i + y_A · \overrightarrow j\) se mai noteaza \(\overrightarrow r_A\)

    Ecuatii trigonometrice fundamentale

    Ecutia \(\sin x = a\) are solutii daca si numai daca \(a ∈ [-1 , 1]\)
    In acest caz solutiile sunt : \(x ∈ {(-1)^{k}arcsin a + kπ / k ∈ N}\)

    Ecutia \(\cos x = b\) are solutii daca si numai daca \(b ∈ [-1 , 1]\)
    In acest caz solutiile sunt : \(x ∈ {±arccos b + 2kπ / k ∈ N}\)

    In acest caz solutiile sunt : \(x ∈ {arcctgd + kπ / k ∈ N}\)
